## Configuración

### Instalar Paquetes

```sh
npm install
```

### Compilación y Hot-Reload para Desarrollo

```sh
npm run dev
```

### Verificar tipos, compilar y minimizar para Producción

```sh
npm run build
```

### Ejecutar Test Unitarios con [Vitest](https://vitest.dev/)

```sh
npm run test:unit
```

### Lint con [ESLint](https://eslint.org/)

```sh
npm run lint
```
